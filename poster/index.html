gEGNO: A Gated Equivariant Graph Neural Operator for Airflow Prediction

GRaM Workshop, Competition Track @ ICLR 2026

Ankit Grover  ·  Rémi Bourgerie

KTH Royal Institute of Technology, Stockholm, Sweden

1Abstract

  • Task. Predict five future velocity frames on a 100k-point cloud surrounding one to three airfoils.
  • Model. gEGNO couples an $E(3)$-equivariant EGNN backbone [3] with FNO-style spectral mixing along time [4] and a Reynolds mean-residual decoder [1].
  • Result. Test L2 = 1.118, reducing the organiser MLP baseline (3.16) by 65% and a persist-last-frame baseline (1.68) by 33%. Ablations at df = 0.15 show spectral temporal mixing as the largest single contribution ($-5.3\%$ test L2), with per-edge gating inside the full gEGNO stack adding a further $-0.7\%$.

2Problem & Symmetries

Given five consecutive frames of the 3D velocity field, predict the next five. Evaluation is mean per-point L2, averaged across the five output frames.

SymmetryConsequence
$E(3)$-equivariance (Navier–Stokes) EGNN scalar/vector separation
No-slip $v=0$ on airfoil Hard zero-masking at airfoil-surface nodes
Reynolds decomposition [1] $v = \bar v + v'$ Predict fluctuation $\Delta v$ about the temporal mean $\bar v$

3Features & Graph

Let $s^\star(x_i) = \arg\min_{s \in \mathcal{S}}\|x_i-s\|_2$ denote the nearest airfoil-surface point.

Truncated UDF $$\tilde d(x_i) = \min(\|x_i - s^\star\|_2,\; d_{\max}),\quad d_{\max}=0.5$$
Inward surface normal (equivariant, $-\nabla \tilde d$) $$\hat n(x_i) = \tfrac{s^\star - x_i}{\|s^\star - x_i\|_2}$$

Fixed kNN (submitted)

Directed $k$-NN with $k = 16$ in $\mathbb{R}^3$, computed per sample via SciPy's cKDTree. The graph topology is invariant under rigid motions.

UDF-adaptive kNN (ablated)

$$k(x_i) = k_{\text{far}} + (k_{\text{near}}-k_{\text{far}})\,e^{-\tilde d(x_i)/\lambda},\ \lambda{=}0.05$$

4Architecture

Time-axis semantics (ours vs EGNO [5])

Ours. The time axis spans five real input frames. Positional indices $0{\dots}4$ serve only as order tags, and the spectral operator mixes genuine temporal history.

EGNO (contrast). A single current state is replicated $P$ times, each copy tagged with its physical future offset $\Delta t_p$. The spectral operator therefore mixes copies of the same state.

TimeConv (spectral mixing along $t$) $$h' = h + \mathrm{LReLU}\bigl(\mathcal{F}_t^{-1}(W^{(k)}\hat h(k))\bigr)$$
TimeConvX (spectral mixing along $t$, applied independently to each velocity component) $$v'_d(x_i,t) = v_d(x_i,t) + \mathcal{F}_t^{-1}(\tilde W^{(k)}\hat v_d(x_i,k))$$
EGNN with multi-head edge gate $$\begin{aligned} m_{ij} &= \phi_e(h_i^l,h_j^l,\|x_i-x_j\|^2,a_{ij}) \\ \textcolor{#c44e52}{e_{ij}^{(h)}} &\textcolor{#c44e52}{= \sigma(\mathrm{Linear}_h(m_{ij})),\; h = 1,\dots,H} \\ m_i &= \textstyle\sum_{j} \textcolor{#c44e52}{\bigl(\tfrac{1}{H}\textstyle\sum_h e_{ij}^{(h)}\bigr)}\, m_{ij} \\ h_i^{l+1} &= \mathrm{LayerNorm}(h_i^l + \phi_h(h_i^l, m_i)) \end{aligned}$$ Edge features $a_{ij}$ (11 scalars): $\|x_i-x_j\|^2$ plus per-timestep projections $\langle v_i^{(t)}, \hat r_{ij}\rangle$ and $\langle v_j^{(t)}, \hat r_{ij}\rangle$ onto the invariant edge direction $\hat r_{ij}$. The red term is our multi-head edge-gate extension of EGNN [3, Eq. 7–8]. The submitted model uses $H = 1$; widening to $H = 4$ regressed test L2 by 1–3%.
Decoder (ours vs EGNN) $$\Delta v_i = \textcolor{#c44e52}{\underbrace{\textstyle\sum_t \alpha_i^{(t)} v_i^{(t)}}_{\text{learned frame blend}}} + \underbrace{\textstyle\sum_j w(m_{ij})(x_j-x_i)}_{\text{EGNN coord update}}$$ The first (red) term is our learned per-frame blend. The second follows EGNN's coordinate update [3, Eq. 4]. All five output frames are decoded in a single pass, without autoregressive compounding.

5Target & Loss

Following Reynolds [1], we predict the fluctuation $\Delta v$ against the temporal mean $\bar v$ of the input window.

$$v_{\mathrm{out}}(x_i,t) = \underbrace{\tfrac{1}{T}\textstyle\sum_{t'} v_{\mathrm{in}}(x_i,t')}_{\bar v} + \Delta v(x_i,t)$$
Per-point MSE $$\mathcal{L} = \tfrac{1}{T}\textstyle\sum_{t=0}^{T-1}\tfrac{1}{N}\textstyle\sum_{i=1}^{N} \|\hat v_{\mathrm{out}}(x_i,t) - v_{\mathrm{out}}(x_i,t)\|_2^{\,2}$$

7Results — Ablations

Each panel isolates one design choice at df = 0.15, where df is the fraction of the 770-file training split used.

  • Graph topology (df = 0.15). Fixed kNN outperforms UDF-adaptive kNN by 1.6% on test L2.
  • Learning rate (df = 0.15). The more aggressive learning rate improves test L2 by 2.6%.
  • Temporal mixing (df = 0.15). Spectral convolution outperforms 1-D time-conv by 5.3%.
  • Residual target (df = 0.15). Predicting the Reynolds mean-residual outperforms the last-frame residual by 0.7%.
  • Gating, in full gEGNO (df = 0.15). The per-edge gate outperforms the ungated variant by 0.73% (1.1151 vs 1.1233).
  • Hidden dim (df = 1.0). Test L2 plateaus at $h = 64$ (1.118). Widening to $h = 96$ lowers val L2 to 0.905 without changing test L2.

9Limitations & Outlook

  • Gating vs attention. The per-edge sigmoid is an unnormalised gate. Unlike GAT's softmax over $\mathcal{N}(i)$ [14], it allows selective muting but has no built-in self-regularisation. Inside the full gEGNO stack, gating reduces test L2 by 0.73%. A direct comparison to SE(3)-Transformer [10] and EquiFormer [11] would test whether softmax normalisation outperforms our unnormalised gate.
  • Methodology caveats. All ablations use a single seed (no confidence intervals) and the 70/15/15 split at df = 0.15. The submission itself was trained at df = 1.0 with a 95/2.5/2.5 split. Effective batch size is 8 via gradient accumulation.
  • Implementation note. The submitted weights ignore the direction of udf_gradient (a bug). Only its magnitude reaches the encoder, and that magnitude is identically 1.
  • Under-explored directions. We fixed depth at $d = 4$ without a principled sweep. The adaptive-kNN scale $\lambda$ could not be swept within GPU memory limits. We did not test equivariant input noise as a symmetry-preserving augmentation.
  • Absolute-coordinate encodings. Only UDF scalars and edge-direction vectors reach the encoder today. Fourier or rotary features on $(x, y, z)$ would expose global position and may help resolve finer spatial structure.
